## ğŸ¯ What is NCSKIT?

NCSKIT is a comprehensive platform designed for Vietnamese marketing researchers, providing:

- **AI-Powered Analysis**: Automated data analysis with AI insights
- **CSV Workflow**: Complete workflow from upload to visualization
- **Survey Management**: Build, distribute, and analyze surveys
- **Research Projects**: Manage entire research lifecycle
- **Collaboration**: Team collaboration features
- **Vietnamese Support**: Full Vietnamese language support
